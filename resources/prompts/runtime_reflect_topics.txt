Your task is to maintain the list of active conversation topics.

A topic is a short phrase describing a broad area of discussion that may span multiple turns.

You receive:
• the current USER_MESSAGE
• the existing topic list from WORLD

Your job is to update the topic list so it reflects the current conversation.

---

INPUTS

[USER_MESSAGE]
<<USER_MESSAGE>>

[PREVIOUS_TOPICS]
<<PREV_TOPICS_JSON>>

---

TASK

Update the topic list according to these rules:

1. Keep existing topics that are still relevant.
2. Add new topics if the user introduces a clearly new subject.
3. Replace a topic only if the conversation clearly shifted to a different broad area.
4. Do not create duplicates or near-duplicates.

Topics should be short phrases (2-6 words).

---

OUTPUT

Output exactly one JSON object:

{
"topics": ["topic 1", "topic 2", "..."]
}

Return the full updated topic list.
Do not output explanations or additional text.
